# Website Not Loading

## Overview

A website not loading typically indicates a failure in DNS resolution, server availability, or application processing. When a visitor requests a page, the domain must resolve correctly and the server must respond with valid content.

## Common Causes

- DNS records pointing to the wrong server
- hosting server outages
- application errors preventing page generation

## How It Is Diagnosed

Typical diagnostics include:

- testing DNS resolution
- checking server uptime
- reviewing error logs
