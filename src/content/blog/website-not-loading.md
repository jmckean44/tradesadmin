# Website Not Loading

## Overview

When a website fails to load, the problem usually originates from DNS resolution, server infrastructure, or application-level failures. A functioning website requires the domain to resolve correctly and the server to respond to incoming requests.

## Common Causes

- DNS records pointing to the wrong server
- hosting server downtime
- web server configuration errors
- application failures within the CMS
- expired or invalid SSL certificates

## How the Problem Appears

- browser displaying server error messages
- visitors unable to reach the website
- inconsistent loading across devices or locations
- website appearing offline

## How It Is Diagnosed

- verifying DNS resolution using diagnostic tools
- testing server availability and uptime
- reviewing server error logs
- inspecting HTTP response codes

## Typical Fix

- correct DNS records pointing to the hosting server
- restore server services or hosting configuration
- resolve application errors within the website platform
- renew or reinstall SSL certificates
