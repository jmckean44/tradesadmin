# Website Slow on Mobile

## Overview

Mobile performance directly affects search rankings and user behavior. Slow pages often cause visitors to leave before interacting with the website.

---

## Common Causes

### Large Images

Uncompressed images are a frequent cause of slow page load times.

Related insight:

- [image-optimization](/insights/image-optimization/)

---

### Excessive Scripts

Multiple tracking scripts and plugins increase page load time.

Related insight:

- [javascript-performance](/insights/javascript-performance/)

---

### Server Performance

Hosting configuration may create delays in page delivery.

Related insight:

- [server-response-time](/insights/server-response-time/)

---

## Quick Diagnosis Checklist

Check these items:

- run a mobile speed test
- check page load time
- inspect image sizes
- review script loading

## Next Steps
