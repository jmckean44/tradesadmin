# Contractor Website Optimization

## Overview

Website optimization involves improving technical performance, search visibility, and user experience. Many contractor websites suffer from hidden issues that limit their effectiveness.

---

## Common Causes

### Slow Website Performance

Large files and inefficient scripts increase page load times.

Related insight:

- [/insights/core-web-vitals/](/insights/core-web-vitals/)

---

### Broken Links

Broken internal links reduce usability and may affect search performance.

Related insight:

- [/insights/broken-links/](/insights/broken-links/)

---

### Improper Redirects

Incorrect redirect rules can prevent pages from loading correctly.

Related insight:

- [/insights/redirect-errors/](/insights/redirect-errors/)

---

### Security or SSL Issues

Security warnings may discourage visitors and affect search visibility.

Related insight:

- [/insights/ssl-errors/](/insights/ssl-errors/)

---

## Quick Diagnosis Checklist

Check these items:

- test website load speed
- check for broken links
- confirm pages load without security warnings
- verify redirects work properly

---

## Next Steps

Optimizing technical performance improves both search visibility and user experience.
